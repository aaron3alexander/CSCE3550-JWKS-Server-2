# CSCE3550-JWKS-Server-2
A JWKS server for CSCE3550 that provides public keys for verifying JWTs, but with a database.

To run:
- Use `npm install`
- Then do `node server.js`

To run tests:
- Make sure that the server isn't already running
- Do `npm run test`

To see coverage:
- Make sure that the server isn't already running
- Do `npm run coverage`
